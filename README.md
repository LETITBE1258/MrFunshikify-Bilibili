# MrBeastify-Bilibili
将随机泛式图片覆盖到B站视频封面的浏览器扩展

在Linux Mint Cinnamon 22.3环境，Firefox 150.0.3和Chromium 148.0.7778.167上测试可用。

🌟**暂时使用方法：** Download ZIP后解压，在 edge://extensions/ 中开启开发人员模式，加载解压缩的扩展，选择解压后的文件夹

CC修改，请不放心使用。
